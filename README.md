# Juego Unity

Proyecto de juego desarrollado en Unity.

## Características Implementadas

- **Pinchos**: Obstáculos colocados en la parte superior e inferior del nivel
- **Sistema de Colisión**: Detección de colisión entre el jugador y los pinchos
- **Escena de Game Over**: Al colisionar con los pinchos, el jugador es llevado a la escena de fin de partida

## Tecnologías

- Unity
- C#

## Instalación

1. Clona este repositorio:
```bash
git clone https://github.com/dloboo/JuegoUnity.git
```

2. Abre el proyecto con Unity

3. Ejecuta la escena principal
